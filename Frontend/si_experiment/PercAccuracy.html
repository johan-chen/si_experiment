{{ extends "si_experiment/Page.html" }} {{ block title }} KI zur Preisvorhersage {{ endblock }} {{ block content }} Wir stellen Ihnen nun eine Künstliche Intelligenz (KI) vor, die darauf trainiert ist, den Verkaufspreis von auf der Plattform gelisteten Immobilien vorherzusagen.

Die KI nutzt die gleichen 11 Eigenschaften von Wohnungen, um den angebotenen Preis vorherzusagen. Die KI basiert auf einem Random Forest, eine der einfachsten, aber auch eine der leistungsfähigsten KI-Methoden. Ein Random Forest nutzt eine Vielzahl unterschiedlicher Entscheidungsbäume, die jeweils einen einzelnen Wert vorhersagen (hier: den Verkaufspreis). Die Mehrheitsvorhersage alle Entscheidungsbäume bestimmt dann die endgültige Vorhersage. Mit anderen Worten: Der Random Forest nutzt die "Weisheit der Menge".

Wie hoch schätzen Sie die Genauigkeit der KI ein, um Immobilienpreise vorherzusagen?
